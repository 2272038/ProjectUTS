Cascade Style Sheet


CSS (Cascade Style Sheet) adalah sebuah bahasa untuk mengatur tampilan web sehingga terlihat lebih menarik dan indah, dengan bantuan CSS, kita dapat mengatur layout (tata letak), warna, font, garis, background, animasi, dan lain-lain.

Syntax

  • Selektor (selector) merupakan kata kunci untuk menentukan elemen HTML yang akan kita beri gaya (style).

  • Blok Deklarasi dalam CSS berisi properti dari Selektor dan nilai dari properti tersebut. Deklarasi diawali dan diakhiri dengan kurung kurawal {...}

  • Properti merupakan atribut yang akan diterapkan oleh elemen yang dipilih (Selektor), untuk mendefinisikan nilai properti, kita perlu menggunakan tanda titik dua (:) setelah penulisan nama properti.

  • Nilai merupakan atribut yang akan digunakan setelah kita menuliskan properti, nilai ini berfungsi untuk menentukan apa yang akan terjadi.

Jenis-jenis CSS

Inline style sheet

Internal style sheet

Eksternal style sheet

  1. Inline style sheet

    2. Inline Style Sheet merupakan metode atau cara penulisan CSS langsung pada tag HTML yang ingin diberikan style CSS. Metode ini biasanya digunakan ketika kita tidak ingin mengatur semua elemen didalam dokumen web. Metode ini sangat tidak disarankan karena akan membuat tag pada HTML akan penuh dengan attribute HTML dan membuat ukuran file pada HTML akan bertambah besar. Berikut contoh penggunaan Inline Style Sheet.


  2. Internal style sheet

    3. Internal Style Sheet merupakan metode penulisan CSS langsung didalam file HTML tepatnya berada diantara tag head dengan diapit tag style terlebih dahulu sebelum memuali kode CSS. Berikut contoh penggunaan Internal Style Sheet.


  3. Eksternal style sheet

    4. Eksternal Style Sheet merupakan metode penulisan CSS secara terpisah dengan memberikan ekstensi .css untuk nama filenya. Metode ini sangat dinajurkan untuk menuliskan kode CSS karena kita akan dengan mudah mengubah kode yang terkumpul dalam satu file. Untuk Eksternal style file akan disematkan atau diload dengan kode HTML seperti berikut.
