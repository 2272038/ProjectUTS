HTML

Apa itu HTML??

HTML atau yang biasa dipanggil HyperText Markup Language ini merupakan bahasa Markup sandart untuk membuat halaman web, HTML menggambarkan struktur web, HTML terdiri dari serangkaian elemen. Elemen HTML memberi tahu browser cara menampilkan konten. Elemen HTML didefinisikan oleh tag awal, konten(isi), tag akhir.

simple HTML document

Penjelasan

  • < !DOCTYPE html > digunakan untuk mendeklarasi jenis dokumen.
  • < HTML > digunakan sebagai element akar dari halaman HTMl.
  • < HEAD > digunakan untuk memberikan informasi tentang dokumen.
  • < TITLE > digunakan untuk menentukan judul pada halaman.
  • < BODY > digunakan untuk memberi isi pada dokumen yang akan ditampilkan oleh web browser.
  • < h1 > digunakan untuk membuat heading dengan ukuran terbesar
  • < p > digunakan untuk membuat paragraf

lainnya

seperti gambar diatas penggunaan tagh1-h6 dengan menambahkan 3 atribute align yang dimulai dari right, center, dan left akan menghasilkan gambar yang terdapat dibelakang.

seperti gambar diatas penggunaan tag P dan juga atribute lain seperti b untuk Bold, i untuk italic, u untuk Underline, sup untuk pangkat keatas, sub untuk pangkat kebawah, s untuk dicoret ditengah, big untuk membesarkan huruf, small untuk membuat huruf terlihat kecil, elemen-elemen tersebut akan menghasilkan gambar yang terdapat dibelakangnya .

seperti gambar diatas penggunaan tag lainnya seperti br sebagai enter, kemudian hr untuk membuat garis, kemudian pre yang mengatur tulisannya sesuai dengan apa yang kita tulis, serta special character dengan awalan &(dan) kemudian isinya yaitu nbsp dan diakhiri dengan ;(titik koma sebagai penutup) .
